Here are some libraries that I use:

1. GLFW - a OpenGL library that helps me with the window, input and events.

2. GLAD - a library that helps me with the shaders, textures, buffers, meshes, models etc.

3. GLM - a math library that contains definitions of vectors and matrices.

4. stb - a OpenGL library that helps me with loading the textures and reading the texture file.

5. nholmann - a OpenGL library that helps me read the mesh files and to interpret them so i can load meshes.

6. assimp - a OpenGL model library loader that helps me to .. Load Models.

7. openal - a sound library that helps with the sound importer, volume changer, the sound effects and many other sound stuffs.

8. entt - a entity library that helps me with the entity characeristics, and to modify the entity.

9. imgui - a GUI library that can run on different graphics API's, from the simple SDL and opengl libraries to the DirectX and Vulkan libraries (which are soooo complicated!).

10 . Imguizmo - a GUI library that helps with the gizmos (which are the rotating, scaling , node graphing etc.) .

11 . Freetype - a library that helps with text rendering on the screen (or the window) .
