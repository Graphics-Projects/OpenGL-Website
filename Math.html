If you want to program in OpenGL, then you have to remember the following math terms:


1. Vectors, which are the most important concept in graphics programming, because you might need to make a set of vertices for the desired object that you want to render.



2. Matrix - a set of vectors - is also important, because when you want to import a camera, then you need to use the MVP matrix, which stands for: Model - View - Projection matrix.
